# Карьерный Компас (Career Wheel)

Интерактивное веб-приложение для оценки карьерного баланса по методике «Колесо баланса». Позволяет анализировать сферы профессиональной деятельности, настраивать цветовую палитру под личные зоны комфорта, вести историю замеров и экспортировать результаты в форматы PNG и PDF.

---

## 🚀 Как запустить проект локально

### 1. Установка зависимостей
Убедитесь, что у вас установлена Node.js (v18+). Установите необходимые пакеты:
```bash
npm install
```

### 2. Запуск сервера разработки
Для запуска приложения в режиме реального времени:
```bash
npm run dev
```
После запуска проект будет доступен по адресу: `http://localhost:3000`

### 3. Сборка для публикации (Production)
Для компиляции оптимизированных файлов проекта в папку `dist`:
```bash
npm run build
```
